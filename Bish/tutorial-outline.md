# Bish Tutorial Outline

特别鸣谢deepseek给我列了这个提纲 :)

## 第一部分：快速入门

1. **安装与运行**  
   - 下载/编译 Bish 可执行文件  
   - 运行 REPL，执行第一个表达式 `print("Hello, Bish!")`  
   - 运行脚本文件 `bish script.bish`

2. **基本语法与数据类型**  
   - 整数、浮点数、字符串、布尔值、`null`  
   - 变量定义（`:=`）与赋值（`=`）  
   - 注释（`//` 和 `/* */`）  
   - 简单输出 `print`

3. **运算符与表达式**  
   - 算术运算（`+ - * / % ^`）  
   - 比较运算（`== != < <= > >=`）  
   - 逻辑运算（`&& || !`）  
   - 空值合并（`??`）

## 第二部分：控制流与容器

4. **条件与循环**  
   - `if` / `else`  
   - `while` / `do while`  
   - `for` 循环（`for (变量 : 可迭代对象)`）  
   - `break` / `continue` / 标签（`tag:`）

5. **列表、映射与对象**  
   - 列表字面量 `[1, 2, 3]`，索引与切片  
   - 映射字面量 `{"a":1, "b":2}`，键值访问  
   - 对象字面量 `{.x:1, .y:2}`（用于快速创建结构）  
   - 内置方法：`add`、`length`、`iter` 等

6. **迭代与生成器**  
   - `iter()` 和 `next()`  
   - 迭代器辅助方法：`map`、`filter`、`reduce`、`toList`  
   - 生成器函数（`func*`）与 `yield`

## 第三部分：函数与模块

7. **函数定义**  
   - 基本函数 `func add(a,b) a+b`  
   - 参数默认值、剩余参数（`..rest`）  
   - 返回值（`return`，无返回则返回 `null`）  
   - 闭包与词法作用域

8. **高阶函数与装饰器**  
   - 函数作为值  
   - 自定义装饰器（`@` 语法）  
   - 内置高阶函数示例（`map`、`filter`）

9. **模块与导入**  
   - `import("module.bish")`  
   - 导入编译后字节码（`.bishc`）  
   - 导入 C# 插件（`.dll`）  
   - `meta.root` 与相对路径

---

## 第四部分：面向对象与模式匹配

10. **类与对象**  
    - 定义类 `class C { ... }`  
    - 构造方法 `init`  
    - 成员变量与方法  
    - 继承（单继承，多重继承通过 MRO）

11. **运算符重载与钩子**  
    - 算术运算符重载（`oper +` 等）  
    - 下标访问（`get []`、`set []`、`def []`、`del []`）  
    - 属性访问钩子（`get`、`set`、`def`、`del`）  
    - 资源管理钩子（`enter`、`exit`，用于 `with`）

12. **模式匹配与解构**  
    - `is` 与模式（常量、类型、`of`、`when`）  
    - `switch` 表达式  
    - 列表/映射/对象的解构赋值（`[a,..b] := list`）  
    - 函数参数的解构

## 第五部分：错误处理与异步编程

13. **错误处理**  
    - `throw` 抛出错误  
    - `try` 表达式（捕获错误为 `ErrorResult`）  
    - `with` 语句（自动资源管理）  
    - 自定义错误类型

14. **异步编程基础**  
    - `async` 函数与 `await`  
    - `Task` 类型：`completed`、`error`、`run`、`sleep`  
    - `Runner.blocked` 阻塞等待结果

15. **组合任务与异步迭代**  
    - `Task.all`、`Task.any`、`Task.merge`、`Task.concat`  
    - `for await (item : asyncIterable)`  
    - 异步生成器（`async*` 与 `yield await*`）

## 第六部分：高级主题与元编程

16. **字节码与虚拟机**  
    - `bytecode` 类型与手动构造指令  
    - `frame` 对象（栈、作用域、指令指针）  
    - 运行时编译（`meta.compile`）

17. **反射与元编程**  
    - 访问对象 `vars`  
    - 动态修改类型 MRO  
    - 使用 `passCaller` 获取调用帧

18. **与 C# 互操作**  
    - 编写简单插件实现 `IPlugin`  
    - 导出函数和类型  
    - 在 Bish 中调用 C# 代码
